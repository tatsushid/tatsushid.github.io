xmllint に標準入力の内容を読ませる

  • bash
2013-12-23 21:49

ときどきAPIなどから返ってきた生の XML を読みたいことがあるんですが、その時、 XML 要素の並びやインデントがきれいに整理されていると、非常に読みやすくていろいろと捗ります。こういう目的のためには xmllint コマンドが便利なのですが、残念ながら標準入力からの読み込みに対応していません。ゆえにパイプなんかも使えず、しばしば困る場合があります。が、 bash を使っていれば、こんな感じで標準入力内容を読ませることができます（ xml_generator.sh は XML を生成するコマンド例）

bash xmllint --format <(xml_generator.sh)

上記例では XML を返すようなコマンドをそのまま指定してますが、ヒアドキュメントを読みたい場合は、

bash xmllint --format <(cat <<_XML_ <foo><bar>baz</bar></foo> _XML_)

みたいに書く事もできます。毎度一時ファイル作る必要もないため、個人的には結構これで助かっています
