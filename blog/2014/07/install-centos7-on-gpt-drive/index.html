CentOS7をGPTパーティションで新規インストールする

2014-07-25
  • centos

CentOS 7 を GPT パーティションでインストールするのは難しくない。 Bug 1032428 - Need a 1MB ‘biosboot’ type partition when install RHEL7 on a GPT disk with custom partition によると、

  • 2TB 未満のディスクを使用すると自動的に MSDOS パーティションを使用
  • 2TB 以上のディスクを使用すると自動的に GPT パーティションを使用

という形となっているため、通常は意識して選択する必要がない。また、 2TB 未満のディスクも GPT パーティションとしてインストールしたければ、インストール CD からインストーラーをブートする時のカーネルオプションに inst.gpt を指定しておくと、ディスク種別にかかわらず GTP パーティションが使われるようになる
